> Free AI-powered image generation, video creation & chat — no login, no subscription, forever free for everyone.

## ✨ Features

- 🖼 "Image Generation" — Powered by Google Imagen 3 (Gemini API)
- 🎬 "Video Generation" — Powered by Wan2.1 & CogVideoX (Hugging Face)
- 💬 "AI Chat" — Powered by Groq (Llama 3.3 70B, Mixtral, Gemma 2)
- 📱 "Mobile Friendly" — Works perfectly on any phone or tablet
- 🚀 "No Login" — Open and use instantly
- 💸 "100% Free" — No subscription, no credit card, no hidden fees

## 🛠 Built With

| Feature | API | Provider |
|---|---|---|
| 🖼 Images | Imagen 3 | Google Gemini |

| 💬 Chat | Llama 3.3 70B | Groq |

## 🚀 How to Use

1. Open the website
2. Choose  "Images", "Videos", or "Chat"
3. Type your prompt
4. Hit "Generate"
5. Download your result — completely free!

---

🔧 Self Host

1. Clone this repo
```bash
git clone https://github.com/jns-pk/JnsGen.git
```
2. Open `JnsGen.html` in any browser
3. Done — no install needed!
